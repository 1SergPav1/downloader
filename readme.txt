ЗАДАНИЕ.
Написать программу, которая открывает файл config.json, содержащий набор ссылок, 
и производит скачивание файлов по этим ссылкам в многопоточном режиме. По мере скачивания 
файлов увеличивать счетчик, хранящий общий объем скачанных файлов в b. Предусмотреть вариант 
решения с синхронизацией через каналы.
КОММЕНТАРИЙ.
Не использовать пакет sync.Waitgroup.
Файлы будем скачивать в директорию с проектом в папку ./download/<дата время>.
